[b]Title:[/b] Textbuilder2e - Convert Pathbuilder exports to readable text (now with Paizo BBCode support!)

[b]Post:[/b]

I made a small Python script that converts Pathbuilder 2e JSON exports into clean, readable text formats. Useful for sharing builds on the Paizo forums, Discord, Reddit, printing character sheets, or just having a plain text backup.

[b]Output Formats:[/b]

[b]Text templates:[/b]
[list]
[*][b]build[/b] (default) - level-by-level progression showing your choices
[*][b]static[/b] - complete character sheet with all stats
[*][b]condensed[/b] - compact format for quick sharing
[/list]

[b]Forum/social formats:[/b]
[list]
[*][b]reddit[/b] / [b]reddit-build[/b] / [b]reddit-static[/b] - Reddit markdown
[*][b]paizo[/b] / [b]paizo-build[/b] / [b]paizo-static[/b] - Paizo forum BBCode
[*][b]bluesky[/b] - ultra-compact for Bluesky
[/list]

[b]Features:[/b]
[list]
[*]Remaster compatible (ability modifiers, no alignment)
[*]Native BBCode output for Paizo forums
[*]Smart word wrap for text formats
[*]Read from clipboard with -c flag
[*]Interactive mode when run without arguments
[/list]

[b]Usage:[/b]

[i]# Generate Paizo BBCode output[/i]
textbuilder2e.py character.json -p paizo-static --stdout

[i]# Generate plain text[/i]
textbuilder2e.py character.json -t static

[i]# Read from clipboard, copy result to clipboard[/i]
textbuilder2e.py -c --copy

Free, open source, no dependencies beyond Python 3.10+.

GitHub: https://github.com/doctor-duus/textbuilder2e

Happy to take feedback if anyone finds it useful!
