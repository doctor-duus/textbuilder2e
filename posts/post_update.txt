**Title:** Textbuilder2e - Convert Pathbuilder JSON exports to readable text and Reddit markdown (update: interactive mode, clipboard support)

**Post:**

Update to my Pathbuilder JSON converter! [Previous post here.](https://www.reddit.com/r/Pathfinder2e/comments/1tc9fce/textbuilder2e_convert_pathbuilder_exports_to_text/)

**New features:**

- **Reddit markdown formats** - Output directly formatted for Reddit posts
- **Interactive mode** - Just run the script with no arguments, it finds JSON files and prompts you
- **Copy to clipboard** - `--copy` pipes output straight to your clipboard (macOS)
- **Auto-load build choices** - If you've run the build format before, it remembers your skill increase selections
- **Spell ranks** - Uses correct PF2e terminology (R1, R2... not L1, L2)
- **Feat notes** - Shows selections like `Assurance [Athletics]` and auto-detects muse choices for `Multifarious Muse [Maestro]`

**Output formats:**

| Flag | Format |
|------|--------|
| `-t build` | Level-by-level progression (default) |
| `-t static` | Complete character sheet |
| `-t condensed` | Compact feats list |
| `-p reddit` | Reddit markdown (compact) |
| `-p reddit-build` | Reddit markdown (full build) |
| `-p reddit-static` | Reddit markdown (character sheet) |
| `-p bluesky` | Ultra-compact for Bluesky |

**Quick usage:**

```
# Interactive mode - finds JSON files, prompts for options
python textbuilder2e.py

# Reddit format, copy to clipboard
python textbuilder2e.py character.json -p reddit-static --copy

# Read from clipboard, output to stdout
python textbuilder2e.py -c --stdout
```
